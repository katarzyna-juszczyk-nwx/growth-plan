# FY27 Growth Plan — Senior Front-End

Working plan for my FY27 competence growth, with an interactive dashboard for
tracking progress through the year.

> **Draft — bound to change.** This is a working plan, not a commitment. Goals,
> metrics, and milestones will be revised as baselines are captured in Q2 and as
> circumstances evolve through the year.

## What's here

| File | Purpose |
|---|---|
| [index.html](./index.html) | Interactive dashboard — tabs per quarter, checkboxes per milestone, progress bars, export snapshot. Open in any browser. |
| [fy27-goals.md](./fy27-goals.md) | One-page summary of all 8 goals — share this with your manager as the committed text. |
| [fy27-goals-detail.md](./fy27-goals-detail.md) | Longer version with specifics, metrics, quarterly milestones, evidence, and risks per goal. |

## Live dashboard

Once GitHub Pages is enabled for this repo, the dashboard lives at:
**https://katarzyna-juszczyk-nwx.github.io/growth-plan/**

The dashboard's progress checkboxes persist in the viewer's browser
(`localStorage`), so each person sees their own view — no server, no sync.

## Structure

- **Horizon:** 2026-04 → 2027-04.
- **Q1 is ramp** (planning, shortlisting, enrolment). Active execution runs **Q2–Q4**.
- **Anchor themes:** AI Engineering Excellence · Frontend Quality & Architecture.
- **Categories:** Results & Outcomes · Behaviors & Impact · Skills & Competencies · Growth & Development.
